## Чек‑лист проверок функции palindrom

### 1. Общие проверки
- Функция вызывается без ошибок  
- Функция всегда возвращает значение  
- Функция возвращает тип — **boolean**

### 2. Корректные палиндромы
- Фраза с пробелами и разным регистром → "А роза упала на лапу Азора" → true
- Числовая строка → "12321" → true
- Одно слово → "топот → true
- Один символ → "." → true
- Пустая строка → "" → true
- Строка с пробелом → " " → true

### 3. Некорректные (не палиндромы)
- Строка без симметрии → "He" → false 
- Числовая строка без симметрии → "123"  → false 

### 4. Невалидные входные данные
- Число → 12321 → false
- null → false
- undefined → false
- Объект → {} → false
- Массив → [1,2,1] → false

### 5. Cтресс‑тесты
- Очень длинный палиндром → 'a'.repeat(10000) → true
- Очень длинный не палиндром → 'a'.repeat(9999) + 'b' → false
