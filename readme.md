## Инструкция по запуску тестов (Jest)

1. Склонируйте репозиторий с проектом:
```bash
git clone https://github.com/dartestqa/palindrom-test.git
```
или скачайте zip‑архив по ссылке и распакуйте его.

2. Убедитесь, что на вашем компьютере установлен Node.js и npm:
```bash
node -v
npm -v
```
Если они не установлены, скачайте и установите Node.js с официального сайта (рекомендуется LTS‑версия).

При установке Node.js автоматически устанавливается npm.
3. Перейдите в корневую директорию проекта:
```bash
cd /путь/до/директории/с/проектом
```
4. Установите зависимости проекта:
```bash
npm install
```
Если Jest не указан в package.json, установите его вручную:
```bash
npm install --save-dev jest
```
5. Убедитесь, что в package.json прописан скрипт для запуска Jest:
```bash
"scripts": {
  "test": "jest"
}
```
6. Запустите тесты:
```bash
npm test
```
или напрямую:
```bash
npx jest
```
7. Для более подробного вывода используйте:
```bash
npm test --verbose
```
8. Для запуска конкретного файла тестов:
```bash
npx jest palindrom.test.js
```
